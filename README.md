# drivers_of_unemployment
As part of a research project, we are examining which indicators drove unemployment pre-2020 and post-2020. 

This project requires a FRED API key.
Create a .env file with:

FRED_API_KEY=your_key_here


Our full results are documented in our paper (titled 'final_result.pdf' in this repo). We'd love to know if you have any questions or suggestions.
